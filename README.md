# 🎬 Movie Search App

ReactとTypeScriptを用いて開発した、リアルタイム映画検索アプリケーションです。
TMDB (The Movie Database) APIを利用し、最新の映画情報や詳細データを取得・表示します。

## ✨ 主な機能
* **リアルタイム検索:** キーワードを入力するたびに即座に検索結果が反映されます。
* **映画詳細表示:** 各映画のポスター、あらすじ、評価などの詳細情報を確認できます。
* **高速な動作:** Viteを採用し、快適なブラウジング体験を実現しています。
* **SPA (Single Page Application):** React Routerを使用し、ページ遷移なしでスムーズに画面が切り替わります。

## 🛠 使用技術 (Tech Stack)
* **Frontend:** React (Hooks), TypeScript
* **Build Tool:** Vite
* **Routing:** React Router
* **API:** The Movie Database (TMDB) API
* **Deployment:** Render (Static Site)
* **Styling:** CSS Modules / Vanilla CSS
